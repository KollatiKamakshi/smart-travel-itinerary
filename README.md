# Smart Travel Project

## Overview
Smart Travel is a web application built using **Next.js** and **React** that allows users to plan and manage travel itineraries efficiently. The app provides an interactive and responsive UI to create, save, and view travel plans.

---

## Features
- Plan and save travel itineraries
- View saved itineraries
- Responsive design for mobile and desktop
- Offline fallback (if no internet connection)

---

## Tech Stack
- **Frontend:** Next.js, React, CSS Modules
- **Backend:** JSON file (local) or optional API
- **Deployment:** Vercel

---

## Local Setup

1. Clone the repository:  

   git clone https://github.com/kollatikamakshi/smart_travel.git
   
3. Install dependencies:

npm install

3.Run locally:

npm run dev
Open the browser at:
http://localhost:3000
